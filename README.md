# Neural Networks From Scratch

A lightweight neural network framework built entirely from scratch using NumPy.

This project explores the fundamental mechanics behind deep learning by implementing forward propagation, backpropagation, optimization, gradient checking, checkpointing, and feature mappings without relying on high-level deep learning libraries.

## Features

* Fully connected neural networks implemented from scratch
* Forward and backward propagation
* Custom activation functions
* Gradient checking using finite differences
* Checkpointing and model persistence
* Early stopping
* Fourier feature mappings
* Polynomial feature transformations
* Image reconstruction experiments
* Training visualization and logging

## Experiments

### XOR Classification

A minimal benchmark used to verify the correctness of the implementation and optimization pipeline.

### Image Classification

Neural networks trained on image datasets using manually implemented training loops and optimization.

### High-Frequency Image Reconstruction

Investigation of Fourier and polynomial feature mappings for learning high-frequency image signals and improving reconstruction quality.
